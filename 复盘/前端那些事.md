​	 parseInt取整(丢掉小数)  math：round四舍五入 ceil向上取整floor向下取整

​	都是es6新增的数据结果 Map值值 有add delete has clear  ；Set键值对 有size set get has delete clear 二者都有keys values entries的遍历/forEach
